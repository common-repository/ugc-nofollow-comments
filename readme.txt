== Description == 

This plugin is used to add &quot;<strong>ugc</strong>&quot; and &quot;<strong>nofollow</strong>&quot; in &quot;<strong>rel</strong>&quot; attribute in external links in comments section. UGC is User Generated Contents which is content created by website users in from of comments or forum posts. 

== Installation ==

To install this plugin follow below steps

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's it, no other settings needed.
